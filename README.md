Como actuaría ante una situación difícil?
Primero escucho bien el problema sin apurarme a responder. Si no tengo la solución inmediata, soy honesta y le digo al usuario que voy a 
investigar y le doy un tiempo estimado de respuesta. Mientras tanto, si hay alguien con más experiencia en el equipo, consulto para resolver más
rápido. Prefiero dar una respuesta correcta un poco después que una apurada que esté mal.
